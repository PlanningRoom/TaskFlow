# Decision 020: Supported Platforms

**Status:** Decided

**Category:** Platform & Access

**Question:** Which platforms will TaskFlow support — web only, or also mobile and desktop?

**Decision:** Web with responsive/mobile-friendly design. A single web codebase optimized for both desktop and mobile screens. No native mobile or desktop apps.

**Rationale:** A responsive web app covers all devices through the browser with minimal extra effort. Modern CSS frameworks make responsive design a standard part of development. Mobile access is a convenience, not the primary use case. PWA or native apps can be explored later if needed.
