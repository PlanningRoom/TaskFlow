# Decision 026: Accessibility Standards

**Status:** Decided

**Category:** Accessibility & Localization

**Question:** What level of accessibility compliance will TaskFlow target (e.g., WCAG 2.1 Level A, AA, or AAA)?

**Decision:** WCAG 2.1 Level AA. TaskFlow will target the industry-standard level of accessibility compliance.

**Rationale:** Level AA is the industry standard and is largely achievable by default when using modern component libraries that are designed to meet this level. It covers the essentials: color contrast, keyboard navigation, screen reader support, and focus management. Targeting AA demonstrates best practices and strengthens the project as a demonstration.
