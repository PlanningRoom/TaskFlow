# Decision 018: Deployment Model

**Status:** Decided

**Category:** Deployment Model

**Question:** Will TaskFlow be deployed as SaaS (cloud-hosted), self-hosted, or both?

**Decision:** SaaS (cloud-hosted). TaskFlow will be deployed and managed as a hosted service.

**Rationale:** For a free demonstration project, SaaS is the simplest path — one version to maintain, full control over infrastructure, and users just access it via the web. Real-time updates from Decision 017 are simpler to deliver with controlled infrastructure. Self-hosting can be offered later if demand arises.
