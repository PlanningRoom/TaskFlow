# Decision 017: Real-Time Collaboration

**Status:** Decided

**Category:** Collaboration Model

**Question:** Are there real-time collaboration needs, such as live updates or simultaneous editing?

**Decision:** Real-time updates. Board changes, status transitions, new comments, and notifications update live without requiring a manual refresh. No simultaneous editing or presence indicators.

**Rationale:** Live updates are the expected baseline for a modern team tool and are necessary for the in-app notifications and activity feed from Decisions 010 and 016 to feel responsive. Simultaneous editing is overkill for task descriptions, and presence indicators can be added later as polish.
