# Decision 010: Must-Have Features

**Status:** Decided

**Category:** Scope & Feature Boundaries

**Question:** What are the must-have features for the initial launch?

**Decision:** Structured MVP. The initial launch will include:
- Projects (create, manage multiple)
- Tasks (create, edit, delete, assign, due dates, priority)
- Task statuses (to-do, in-progress, done)
- Labels/tags on tasks
- Comments on tasks
- Notifications
- Activity feed
- Filters and search
- Basic dashboard/overview
- Invite system for user onboarding

**Rationale:** A medium team of 10-50 users needs enough structure to manage real work across multiple projects. This feature set provides genuine utility — projects, tags, filters, search, and a dashboard give teams visibility — without over-building into subtasks, dependencies, or reporting, which can come in later iterations.
