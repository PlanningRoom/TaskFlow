# Decision 025: Data Residency

**Status:** Decided

**Category:** Data & Compliance

**Question:** Are there data residency requirements that dictate where user data must be stored?

**Decision:** Single region with documented location. Deploy to one cloud region and clearly state where data is stored in the privacy policy.

**Rationale:** Aligns with the GDPR-aware approach from Decision 024 — transparency about data location is a basic privacy practice. The invite-only user base is small enough that a single region provides acceptable performance. Multi-region can be explored later if needed.
