# Decision 006: Logo & Wordmark

**Status:** Decided

**Category:** Visual Identity & Brand

**Question:** What logo and wordmark treatment should TaskFlow use in the sidebar, header, and login screens? Even as a demonstration project, the mockups need a visual identity anchor.

**Decision:** A simple text wordmark using Inter in semibold weight, with a small geometric icon mark to the left. The icon mark is a minimal abstract representation of a checkmark or task flow (e.g., a rounded square with a check, or overlapping shapes suggesting movement/progress). The mark uses the primary indigo accent color; the wordmark text uses the base dark color. On the collapsed tablet sidebar, only the icon mark is shown.

**Rationale:** A text-based wordmark keeps the identity simple and avoids the need for custom illustration work on a demonstration project. Using the same typeface as the UI (Inter) creates cohesion. The small icon mark gives the brand something recognizable when the sidebar collapses to icons on tablet. Keeping the mark geometric and abstract ensures it scales well at small sizes and doesn't look dated.
