# Decision 008: File Attachments

**Status:** Decided

**Category:** Task Features

**Question:** Can users attach files to tasks or comments, or is TaskFlow text-only? If attachments are supported, what types and size limits apply?

**Decision:** No file attachments at launch. Tasks and comments are text-only.

**Rationale:** File attachments require cloud storage infrastructure, upload/download UI, file type validation, size limits, virus scanning considerations, and storage cost management — significant complexity and potential cost for a demonstration project with no budget for paid third-party services. Teams can share files via external tools and reference them in comments with links. Attachment support can be added in a future iteration.
